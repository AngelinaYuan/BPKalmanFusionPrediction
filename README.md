# 项目简介：
    > 在运动目标的，最常见的问题包括：光照变化、运动目标发生形变、目标被遮挡等。本项目针对运动目标被遮挡，提出BP神经网络+Kalman滤波器的联合预测模型。传统的卡尔曼预测的基本思想是：构建状态方程，由当前时刻的位置计算出下一时刻目标的位置（状态值），同时，将当前时刻的位置作为下一时刻的观测值，然后将两者的值融合，得出最优估计值，即卡尔曼滤波器预测出的目标位置。
